Making audio samples
-----------------------------------
1. Record the sample/s
2. Using an audio tool (like Audacity), isolate a sample,
   single wave form recommended to save space.
3. Save samle as 16bit PCM WAV, stereo or mono as desired
4. Convert WAV file to C/C++ header
   Using HxD Hex Editor   https://mh-nexus.de/en/
   a. open WAV file
   b. File->Export->C
   c. save file
   d. copy, include, etc, into project
